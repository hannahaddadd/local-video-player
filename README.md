# Local Video Player

When your local media player fails you, check if this web-based media player does the job.

*Made with AI on a coffee break.*

---

## Features

- Video & audio playback (mp4, mkv, webm, mp3, flac, and more)
- SRT & VTT subtitles with real-time delay adjustment
- Playlist with shuffle and loop modes
- Resume playback position per file
- Waveform visualizer for audio files
- Theater mode
- Two themes: Stylish and Retro
- Keyboard shortcuts

## Built with

- [Video.js](https://videojs.com/) — HTML5 video player

## Usage

Open `index.html` directly in your browser — no server needed.
